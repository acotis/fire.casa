Projects

  • Perplexions — a puzzle game I wrote by hand in pure Rust in just ten days. Probably my favorite single project I've completed.
  • One Down, a simple utility to convert a markdown-like format for describing crossword puzzles into a PNG.
  • clubs-diamonds — an expression brute-forcer for code golf. It's a Rust crate! I have saved many bytes in code golf using this, and a few other people have saved a few bytes.
  • flower-pot — a simpler Rust crate that just gives short handles to all the ANSI color-codes and styling-codes. I wrote this because I'm too lazy to use any of the ones that make you learn a paradigm of string.color(Color::RED) -> ColoredText.
